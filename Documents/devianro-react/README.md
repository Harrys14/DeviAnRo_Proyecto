# DeviAnRo - Delivery de Comida

Aplicación web de delivery de comida desarrollada con **React + Vite + Clerk** para autenticación.

## Funcionalidades

- Autenticación completa con **Clerk** (Login, Registro, Logout)
- Catálogo de productos con búsqueda y filtros por categoría
- Carrito de compras con persistencia (localStorage)
- Página de detalle de productos
- Perfil de usuario
- Diseño responsive

## Tecnologías

- **React 18** + Vite
- **Clerk** - Autenticación
- React Router DOM
- LocalStorage para carrito
- Strapi (Backend)
